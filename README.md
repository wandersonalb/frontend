FRONTEND APLICAÇÃO DE USUÁRIOS

EXECUTAR NO LINUX
-------------------
```
git clone https://github.com/wandersonalb/frontend
```
```
sudo apt install nodejs
```
```
sudo apt install npm
```
```
cd frontend
```
```
npm install
```
```
npm start
```

EXECUTAR NO WINDOWS
-------------------

Download [Node e NPM](https://www.guru99.com/download-install-node-js.html)

```
git clone https://github.com/wandersonalb/frontend
```
```
cd frontend
```
```
npm install
```
```
npm start
```
